# Seminarska-naloga-UPO
Kako sem izdelal seminarsko nalogo pri predmetu UPO (Upravljanje s podatki).

Imeli smo za izdelati konceptualni logični in fizični model pri predmetu UPO in sicer na temo enega problema iz relanega življenja. Jaz sem si izbral poslovanje avtosalona. Naloga vključuje opis problema, uporabniške zahteve, izdelavo konceptualnega ER-modela, pretvorbo v logični model ter predstavitev primerov zapisov za vse tabele. Posebna pozornost je namenjena pravilnemu definiranju razmerij 1:n in n:m ter skladnosti tujih ključev s primarnimi ključi.

Naloge sem se lotil v aplikaciji DIA portable v kateri sem izdelal konceptualni model, ki je vseboval tabele: kupec, račun, prodajalec, salon, avtomobil in pošta. Imel sem še vmesno tabelo med računom in avtomobilom z imenom postavke.
